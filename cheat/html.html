My cheat sheet to basic commands in web development(html)

Basics

All commands need closing tag(</>) unless otherwise stated

Attributes are assigned in opening brackets eg <div class="header">

other attributes, href=(path/url), id=#unique-identifier, title="mousover-text".

syntax name description
<!DOCTYPE html> declares doc type declares file as html to browser does not need closing brackets
<html> starts and ends html everything between is html
<head> starts head section the head section is for meta data not shown on actual webpage eg. title
<body> starts body everything in the body ection is the content actually on the webpage
<header> <main> <footer> <div> starts different sections sections to easily read html. div is generic
<h1><h2>...<h6> different heading sections should have only one h1 section per page.
<p> paragraph usually contains the main body of text on the page
<a> link makes a link externally or internally
<br> line break makes a line break anywhere
<span> span spans are kept in line whit the rest of content. Used to make classes or id's for formatting in CSS or links etc
<button> makes a button.. probably has a js action or link associated

Media

Images

<img src="url.com/dumbimage.jpg width=123px>

Images do not need a closing tag. Atrributes in opening tag. Must have src=url. can also be used to link to a relative path <img src=../media/dumbumage.jpg> Can have attributes such as sizing in pixels and % You can also nest it inside of an <a href=> tag to make the image into a link. Another common attribute is alttext to make a text if the image is not loaded.

Video

<video controls src="source-url.com">
</video>

can also use a <src="url> instead of the src attribute in the opening tag

Embed

<embed src=url>

basically the same as video but can accept gifs and pics etc

Tables

syntax name description
<table> starts table all table elements within these tags
<thead> <tbody> starts table head and body head usually has the column names and body the actual table
<tr> table row starts new row in table (even the head)
<td> table data this is actually contents of individual cell
<colspan> <rowspan> spans how many rows or columns the cell can take up if >1
