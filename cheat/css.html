Basics

All commands need a selector:X and atrributes in the property:value format

x{
property:value;
}

h1{
color:pink;
}

The CSS sheet must be linked to in the html head such as:

<link rel="stylesheet" href="./resources/css/stylesheet.css">

Different kinds of Selectors

*{
font-family:Helvetica;
}

The * selector will select every type of element on the page. In this example making all text Helvetica

.main{
background-color:pink;
}

The .class selector will select everything with the class= attribute in html. In this example everything assigned the main class will have a pink background.

#ID{
color:pink;
}

The # atribute will select everything with id set in html

h1, .example{
color:pink;
}

Using commas like this example will just select multiple things and setting the attributes to all selectors

.example h1{
color:pink;
}

Not using commas it selects nested items. In the example above it would select the h1 componenets within a .example class

ul{
padding:20px 10px 5px 0px;
}

ul{
margin:20px;
}

ul{
padding:20px 10px;
}

ul{
margin:20px 10px 5px;
}

In the margin and padding attributes if you use one value like the second example it will apply to all 4 sides. If you use all 4 it applies in the top, right, bottom left order. like NESW compass order. If you use 2 values it will apply to the vertical ones and then the horizontal values. And three values will apply in the order top, sides, bottom.

a:hover{
color:orange;
}

You can use the psuedo-class hover to change attributes when the mouse is over the property other pseudo-classes are :link, :hover, :active, and :visited

Box Model

MDN Box model page

flexbox

div{
display:inline-flex;
justify-content:space around;
align-items:baseline;
}

flexbox is the parent item with a display of inline-flex, justify content is left to right alignment and align-items is vertical alignment

flex-basis is the value(px, %?) that the item will be before shrink or grow. flex-grow is how much a flex item should strech to fill width proportionatally, with 0(default) being none of course. flex-shrink is obviously the opposite with default of 1.

flex: is a shorthand for grow/shrink/basis in that order. flex: 2 1 200px; will gow at a rate of 2 shrink at 1 and have a basis of 200 px.
Minimum and max values will take precedence over flex properties.

flex-wrap can be used to move flex items to the next line if needed. wrap will move them to the next line if needed.
wrap-reverse will move the 1st items down making a row above to wrap to. no-wrap is default

align-content can be used to space rows within a flexbox

flex-direction default is rows. If you give the direction a value of column it will default to columns instead. Can also take row-reverse and column-reverse

flex-flow can be used to shorthand flex-direction and flex-wrap eg. flex-flow:column wrap

Grids

Can make a parent object a grid by using display:grid
Can make grid columns and rows set with grid-template-columns and grid-template-rows:50px 50% 1fr; fr units are just shorthand for fractional units. For instance 1 fr 2fr 1fr is equal to 1/4 2/4 and 1/4. the fr units will take up fractions of the remaining space if used in conjunction with px or % units.
You can combine rows and columns with grid-template: 1fr 50% 1fr / 2fr 1fr 1fr; everything before the / is rows and after the / is columns.

Can add gaps using column-gap, row-gap and gap gap combines them into one attribute with rows first gap: 20px 5px: will put 20px gap between rows and 5px between columns

children attributes

Can specify the number of rows/columns a child takes up using grid-column-start, grid-column-end, grid-row-start, and grid-row end These can be shortened to grid-row:2 5; or using span:grid-column:2 span 3; and these can be shortened using grid-area: 2 / 2 / 5/ span 3;
This has the order row-start / column-start / row-end / column end.
