My cheat sheet to basic commands in web development(CSS)

Basics

All commands need a selector:X and atrributes in the property:value format

x{
property:value;
}

h1{
color:pink;
}

The CSS sheet must be linked to in the html head such as:

<link rel="stylesheet" href="./resources/css/stylesheet.css">

Different kinds of Selectors

*{
font-family:Helvetica;
}

The * selector will select every type of element on the page. In this example making all text Helvetica

.main{
background-color:pink;
}

The .class selector will select everything with the class= attribute in html. In this example everything assigned the main class will have a pink background.

#ID{
color:pink;
}

The # atribute will select everything with id set in html

h1, .example{
color:pink;
}

Using commas like this example will just select multiple things and setting the attributes to all selectors

.example h1{
color:pink;
}

Not using commas it selects nested items. IN the example above it would select the h1 componenets within a .example class

ul{
padding:20px 10px 5px 0px;
}

ul{
margin:20px;
}

ul{
padding:20px 10px;
}

ul{
margin:20px 10px 5px;
}
