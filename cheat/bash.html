My cheat sheet to basic commands in web development

bash(windows) commands

syntax name description
mkdir make directory makes new folder/directory
ls list list files and folders in current directory
pwd print working directory returns path to current directory
cd change directory changes current folder/directory to a new one relatively eg cd ../resources/media
touch make new file makes file. can be .txt, .html, .css etc
cp copy directory/file copies folder/file to new location. cp index.html about.html
